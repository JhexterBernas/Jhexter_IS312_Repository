# JhexterBernas_hello_world
This project is created using python for IS 312 Activity 3. When accessed through a browser and the program runs, it displays "Hello World! Jhexter S. Bernas" message.

# FILE STRUCTURE

Flask - a simple python tool used to build for building web applications.

# HOW TO RUN APPLICATION LOCALLY

Install Python

Open terminal in the project folder

Make sure flask is installed

Run the application app.py file

Open a web browser and go to http://127.0.0.1:5000/

The message "Hello World! Jhexter S. Bernas" will appear
